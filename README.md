# 📒 Contact Book - C++ OOP Project

A simple contact management system built with C++ using Object-Oriented Programming principles.

## ✨ Features

- ➕ Add new contacts (name, phone, email)
- 📋 View all contacts  
- 🔍 Search contacts by name
- ✏️ Edit contact information
- 🗑️ Delete contacts
- 💾 Persistent storage (saves to file)

## 🏗️ Project Structure

| File | Description |
|------|-------------|
| `Contact.h` | Contact class declaration |
| `Contact.cpp` | Contact class implementation |
| `ContactBook.h` | ContactBook class declaration |
| `ContactBook.cpp` | ContactBook class implementation |
| `main.cpp` | Main program with menu |

## 🚀 How to Compile and Run

```bash
# Compile
g++ Contact.cpp ContactBook.cpp main.cpp -o ContactBook

# Run
./ContactBook
